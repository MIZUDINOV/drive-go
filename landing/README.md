# Drive GO Landing

Static landing page for `drivego.app`.

## Files

- `index.html` - page markup
- `styles.css` - visual style and animations
- `script.js` - scroll reveal and footer year

## Local preview

Open `index.html` in the browser, or run a local static server.

## Before publishing

1. Replace Chrome Web Store links in `index.html` with the real listing URL.
2. Update support email if needed.
3. Add `privacy` and `support` pages for Chrome Web Store fields.
4. Deploy folder contents to Cloudflare Pages.
